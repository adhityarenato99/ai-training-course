Basic Level Program

AI Ready: Manufacturing Fundamentals for Smart Industry

Program Overview

This 9-session foundational program prepares manufacturing professionals for AI implementation by developing core math, programming, and AI concepts. It introduces custom GPT and Gemini fine-tuning, prompt engineering, and basic RAG construction with workflow automation, enabling participants to build simple, practical AI tools.

Learning Goals

  • Understand AI concepts & their manufacturing applications.
  • Acquire Python programming basics for AI tasks.
  • Perform basic fine-tuning and system instruction customization of GPT/Gemini.
  • Build simple RAG systems using manufacturing data.
  • Implement introductory AI-driven workflow automations.

9-Session Curriculum

  • Introduction to AI in Manufacturing: AI concepts, impact, and custom LLMs.
  • Essential Mathematics: Algebra, functions, and statistics with manufacturing examples.
  • Statistics & Probability: Descriptive stats and quality control use cases.
  • Linear Algebra Foundations: Vectors, matrices, and their role in AI.
  • Python Basics & System Instructions: Python setup and crafting instructions for fine-tuning.
  • Programming Control Flow & Functions: Conditionals, loops, and automation scripting.
  • Data Preparation for Fine-Tuning: Data collection, annotation, and cleaning.
  • Intro to ML & Prompt Engineering: ML types and prompt patterns.
  • Custom RAG & Workflow Automation Basics: Building simple RAG with LangChain & Make.com.

Assessment & Scoring

  • 25% - Weekly Quizzes
  • 30% - Programming Assignments
  • 20% - Mid-term Project (Dataset prep & fine-tuning)
  • 20% - Final Project (Custom RAG & automation demo)
  • 5% - Participation

Program Summary

Focus Area Activities Tools & Platforms
Fine-tuning, system instructions, basic RAG setup Dataset prep, prompt engineering, simple workflows ChatGPT API, Gemini, LangChain, Make.com

Final Project: Smart Defect Troubleshooter

Description

Participants build a mini AI assistant that uses a RAG system connected to a local CSV file of defect types and solutions. It integrates a customized Gemini model and connects to Make.com to automate alert emails for high-severity defects.

Advanced Level Program

AI Implementation Mastery: From Data to Deployment

Program Overview

An advanced 12-session program focused on mastering large language model (LLM) customization for manufacturing. Participants will engage in hyperparameter tuning, full model training, advanced prompt engineering, sophisticated RAG builds, and production-grade workflow automation integrating GPT/Gemini within complex industrial systems.

Learning Goals

  • Perform hyperparameter tuning & advanced model customization.
  • Conduct end-to-end training of LLMs with domain-specific datasets.
  • Create robust, scalable RAG systems for knowledge management.
  • Build AI-powered, automated workflows for decision-making.
  • Lead AI implementation and deployment projects in manufacturing.

12-Session Curriculum

  • Foundations of Industrial AI: Refresher on AI/ML and custom model capabilities.
  • Industrial Data Collection & Management: Advanced sourcing, IoT integration, quality assessment.
  • Data Preprocessing & Augmentation: Large-scale dataset prep and tokenization.
  • Feature Engineering & Advanced Prompt Architecture: Domain-specific features, complex prompt chaining.
  • Hyperparameter Tuning & Training Strategies: Learning rate, batch size, epochs, regularization.
  • In-Depth Prompt Engineering: Chain-of-thought, multi-modal prompts, iterative optimization.
  • Machine Learning Model Optimization: Performance metrics, model debugging, error analysis.
  • Deep Learning & Full LLM Training: Infrastructure, transfer learning, handling large data.
  • Predictive Maintenance & Quality Control: Deploying models for anomaly detection.
  • Process & Supply Chain Optimization AI: AI in scheduling, logistics, energy management.
  • Deployment, Scaling & Workflow Automation: Integrating LLMs/APIs with Make.com.
  • Final Project Presentations & Future Directions: Demos, peer review, and emerging trends.

Assessment & Scoring

  • 30% - Weekly Assignments (Coding, tuning, prompts)
  • 20% - Mid-term Evaluation (Technical test)
  • 40% - Final Project (Full custom LLM + RAG + workflow)
  • 10% - Participation & Q&A

Program Summary

Focus Area Activities Tools & Platforms
Hyperparameter tuning, full model training, robust RAG LLM training, advanced optimization, RAG scaling GPT-4 API, Gemini AI Studio, LangChain, Make.com, AWS/GCP

Final Project: Industrial Knowledge Assistant

Description

An enterprise-grade AI assistant powered by a fine-tuned LLM trained on factory maintenance logs and SOPs. It features real-time search, decision support via prompt chaining, and connects to Make.com for full process automation.
